Gramática

S ::= a A
A ::= b B d | c C | d D
B ::= a C | b B
C ::= c S | ε
D ::= a b

FIRST

S: { a }
A: { b, c, d }
B: { a, b }
C: { c, ε }
D: { a }

FOLLOW

S: { $, d }
A: { $, d }
B: { d }
C: { $, d }
D: { $, d }
